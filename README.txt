The website for eCom Gurus final project is https://github.com/ArtinBog/ecom_guru

The website for eCom Gurus screencast  is https://youtu.be/gKF-8lk6Fc4

The homepage is index.html. The folder structure is standard with html files in the root and subfolders for css,
data, imgs, and js.

We used font libraries from Fontawesome and Google.

There are no non-obvious features of the website. Explanations and interactivity are described in the brief at the
top of each page.
